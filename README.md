# puddle

A resource pool manager for gleam

## Quick start

```sh
gleam test  # Run the tests
gleam shell # Run an Erlang shell
```

## Installation

This package can be added to your Gleam project:

```sh
gleam add puddle
```

and its documentation can be found at <https://hexdocs.pm/puddle>.
